# 실시간 광고 입찰 시스템 (Real-Time Bidding System)

## 프로젝트 개요
실시간 광고 입찰(RTB) 시스템은 광고주와 매체사 간의 효율적인 광고 거래를 지원하는 플랫폼입니다. 이 시스템은 실시간 입찰, 타겟팅, 광고 최적화 등의 기능을 제공하여 광고 효율을 극대화합니다.

### 주요 기능
- 실시간 입찰 처리 (Real-Time Bidding)
- 광고 타겟팅 및 세그먼트 관리
- 광고 캠페인 관리
- 실시간 성과 분석 및 리포팅
- 예산 관리 및 입찰 최적화
- 광고 소재 관리

## 시스템 아키텍처

### 핵심 컴포넌트
1. **API 서버**
   - 광고주/매체사 인터페이스
   - 캠페인 관리
   - 입찰 요청/응답 처리

2. **실시간 처리 엔진 (Flink)**
   - 입찰 스트림 처리
   - 실시간 데이터 집계
   - 성과 분석

3. **Worker 서비스**
   - 배치 작업 처리
   - 데이터 동기화
   - 리포트 생성

4. **데이터 파이프라인**
   - Kafka 기반 메시지 처리
   - 실시간 데이터 스트리밍
   - 로그 수집 및 분석
